================================================================================
Cloning the repository
================================================================================

    $ git clone https://bitbucket.org/sdlu/sdlu SDLU

================================================================================
Pulling new versions
================================================================================

    $ git pull origin master

================================================================================
Checking out tagged releases/snapshots
================================================================================

    $ git checkout release-[version]
    $ git checkout snapshot-[date]

    $ git checkout release-latest
    $ git checkout snapshot-latest

================================================================================
Listing tags
================================================================================

    $ git tag -n
